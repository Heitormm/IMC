# IMC
Calculadora de IMC (Índice de Massa Corporal) desenvolvida em Python. O sistema permite calcular o IMC a partir do peso e altura informados pelo usuário, exibindo a classificação correspondente de acordo com os padrões de saúde. Projeto criado para praticar entrada de dados, operações matemáticas e estruturas condicionais em Python.
